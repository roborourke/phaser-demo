Phaser Demo

Demo game put together in stages as part of an introduction to @photonstorm's phaser.js at Liverpool JS User Group

Installation

  1. Clone the repository
  2. Run npm install
  3. Run grunt connect
  4. Browse the examples at http://localhost:8005/indexXX.html
